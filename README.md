
# Portfolio Website – Mithanya Murugesan

## Overview
This is a personal portfolio website built using HTML, CSS, and minimal JavaScript. The project is designed to showcase my skills, projects, and contact information in a clean and structured layout.

## Objective
To create a responsive and professional portfolio that highlights:
- Personal profile
- Technical skills
- Project showcase
- Contact details

## Technologies Used
- HTML5
- CSS3
- JavaScript (basic interaction)

## Features
- Fully responsive design
- Clean and modern UI
- Smooth scrolling navigation
- Organized sections (Home, Projects, Contact)
- Simple and lightweight structure

## File Structure
This project is built as a single file:
- index.html → Contains HTML structure, CSS styling, and JavaScript functionality

## How to Run
1. Download the project
2. Open the `index.html` file in any modern web browser
3. No additional setup required
